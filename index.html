Outreach Plan

Ziel: passende Creator finden, fair verhandeln, langfristig denken.

  1. Research: Shortlist (20–30 Creator), Audience-Fit prüfen
  2. Kontakt: persönliches Intro + Value + klare Erwartung
  3. Call: Tonalität, Grenzen, Timing, Kosten, Deliverables
  4. Deal: Paket + Tracking (Code/UTM) + Nutzungsrechte klären

Briefing & Brand Safety

Ziel: klarer Rahmen, aber genug Freiheit für authentischen Content.

Must-have (immer)

  • CHEEX Positionierung: ethisch, inklusiv, consensual
  • Hook in den ersten 2 Sekunden (Story/Reel)
  • Klare CTA: „7 Tage kostenlos testen“
  • Tracking: Code + Link (UTM)

Do’s & Don’ts

  • Do: Education, persönliche Perspektive, respektvolle Sprache
  • Don’t: explizite Bilder, vulgäre Begriffe, Clickbait
  • Don’t: Druck, Shaming, „Porno“-Framing
Abnahme: Story-Text/Slides vorab kurz prüfen (schneller Prozess, kein Over-Control).
